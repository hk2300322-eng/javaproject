📌 Description

This is a simple console-based Library Management System built using Java. It allows users to manage books using basic operations like search, issue, return, and display books. The project is developed using Object-Oriented Programming (OOP) concepts.

🚀 Features
Display all books
Search book by title
Issue a book
Return a book
Track availability status

🛠️ Technologies Used
Java
OOP Concepts
Arrays
Scanner Class

▶️ How to Run
javac project4.java
java project4

📂 Project Structure
Book Class → Stores book details
Library Class → Handles operations
Main Class → Menu-driven interface

🎯 Learning Outcome
OOP implementation in Java
Class and object usage
Basic library system logic
